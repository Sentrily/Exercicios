Ataques Cibernéticos Descrição
Ransomware Um ataque cibernético no qual os invasores bloqueiam o acesso aos seus dados ou sistemas da vítima, exigindo o pagamento de um resgate para restaurar o acesso.
Phishing Uma técnica de engenharia social usada para enganar usuários de internet usando fraude eletrônica para obter informações confidenciais
DDoS Um tipo de ataque cibernético que tenta indisponibilizar um website ou recurso de rede inundando-o com tráfego mal-intencionado e deixando-o incapaz de operar.
Malware Um software malicioso projetado para infiltrar, danificar ou controlar sistemas e dispositivos.
